# Playment Assingment

## Instalation

Run these commands
```
git clone git@github.com:satish876/playment-assignment.git
cd playment-assignment
npm i && npm start
```

## Available commands

1. cd path
2. mkdir path
3. rm path

  Note: path can be absoulte as well as relative

4. ls
5. pwd
6. session clear
7. .clear or clear

    to clear the terminal
